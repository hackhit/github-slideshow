# Security Policy

## Supported Versions

Since this is a personal project used for generating profile assets, only the latest version of the code in the `main` branch is supported.

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do not report it publicly. 

### How to Report
Please email the repository owner at the email address associated with their GitHub profile (or open a private advisory if enabled).

### Response
I will attempt to respond to your report within 48 hours.
